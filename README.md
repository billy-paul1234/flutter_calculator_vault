# Flutter_calculator_vault

This app has a Secret File system inside a Calculator. 

## Developing Stage 

[+]--Multiple options to manage the Files inside the Vault.

## To install and run it:

### `flutter pub get`
To install the required packages.

### `flutter run`
To run the project
